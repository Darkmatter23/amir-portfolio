# Amir Goudarzi — Portfolio

Personal portfolio site built with React, deployed on Vercel.

## Run locally

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Deploy

Connected to Vercel — every `git push` to `master` auto-deploys.

## Update content

Edit the `EXPERIENCE`, `SKILLS`, and `EDUCATION` arrays at the top of `src/App.js`.
